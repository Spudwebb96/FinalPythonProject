## Présentation générale :

Lors de ce projet, nous avons réalisé un jeu video en python a l'aide du module pygame.

Ce jeu prend la forme d'une fenêtre de résolution 1440*1024, contenant un menu principal, un menu de choix de personnages pour enfin arriver en jeu.

Pour plus d'informations : 
https://github.com/Symchowicz/FinalPythonProject

## Présentation des rêgles du jeu :

Bienvenue dans Words Legends !
 Affrontez vos amis dans un duel de phrases.Le but est de créer la phrase qui rapportera le plus de points. 
 Vous gagnez des points si votre phrase est grammaticalement correcte :

                Sujet + Verbe + Complément
                          (+ mots de liaisons si nécessaire) 

Attention au code couleur !
 Les mots sont classés par code couleur :
Sujets > Rouge / Verbes > Bleu / Compléments > Vert / Liaisons > Jaune

Vous devrez alors composer votre phrase avec 1 Sujet, 1 Verbe, maximum 2 Compléments et maximum 1 Liaison.
